# GeckoLib使用笔记

对于Java原版，如果没有特殊的技巧，想要实现动画效果并不容易。但在基岩版中，已经有成熟的动画系统。为了在Java版中使用这一系统，那么就添加 ***GeckoLib*** 作为依赖吧，配合 ***BlockBench*** 制作模型与动画，实现各种你所能想到的内容。
