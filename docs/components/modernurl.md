# ModernUrl
现代化的超链接,从`GitBook`模仿而来

## 使用
```html
<ModernUrl icon="/icon/github.png" 
title="GitHub - neoforged/MDK: The Mod Developer Kit" 
stitle="Github" 
desc="示例"
url="https://github.com/neoforged/MDK" />
```
